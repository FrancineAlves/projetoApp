git init  - monta o git na pasta atual 
git status - mostra status arquivo
git add Arquivo    - adiciona arquivo para proximo commit commitar
git add --all   - adiciona todos os arquivos


git commit  -m “Cria o arquivo exemplo”     - atualiza arquivos para commitar (-m parametro adicionar mensagem)


toda alteracao deve ser definida para proximo commit salvar, com git add Arquivo, se nao não salva!

git commit  -a -m “adicionar arquivos em para commitar e comittar em sequencia”
o arquivo já deve ter sido commitado pela primeira vez.



git branch v1   - cria branch v1

git checkout v1  - seleciona o branch v1

git merge <selecionar merge para juntar>
